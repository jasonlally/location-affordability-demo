Location Affordability Index Search

There is more to housing affordability than how much rent or mortgage you pay. Transportation costs are the second-biggest budget item for most families, but to date there hasn't been an easy way for people to fully factor transportation costs into decisions about where to live and work. The Location Affordability Index was built by HUD and its partners to capture the true cost of housing decisions. This is a demo application built on top of a prototype API. You can learn more about the API here. Currently, you can only search addresses in the Denver-Aurora-Broomfield MSA (Census region).
